# AV Media Telangana Broadcast Control Panel v1.0

A dedicated, lightweight live production control panel built for **OBS Studio Custom Browser Docks** and standalone browser operation.

---

## 🛠️ OBS Studio Integration (Custom Browser Dock Setup)

1. Launch **OBS Studio**.
2. Go to **View** -> **Docks** -> **Custom Browser Docks...**
3. Under **Dock Name**, enter: `AV Media Control Panel`
4. Under **URL**, select local file path:
   `d:\AV Media Telangana\av-media-telangana-broadcast-kit\control-panel\index.html`
5. Click **Apply** and dock the panel window anywhere in OBS Studio interface!

---

## ⚡ Live Workflow (Preview & Go Live)

1. **Type Headlines**: Enter news items line-by-line in the text box.
2. **Select Parameters**: Choose Category (`రాజకీయం`, `క్రీడలు`, `వాతావరణం` etc.), Theme (`Default` / `Breaking`), and Speed.
3. **Preview**: Click `👁️ Preview` to see the live rendering box update inside the Control Panel.
4. **Go Live**: Click `🔴 GO LIVE (ON-AIR)` to push the update instantly to the OBS Ticker Browser Source without restarting or manual editing!
